# Splunk Failed Login Dashboard

📊 A simple Splunk dashboard to monitor failed login attempts using a sample dataset.  
This project demonstrates multiple Splunk visualizations built for security monitoring.

---

## 🚀 Features
- **Radial Gauge** → Shows total failed logins
- **Bar Chart** → Failed logins by user
- **Line Chart** → Failed logins over time
- **Area Chart** → Trend comparison
- **Statistics Table** → Raw data in tabular format

## 🛠️ How to Use
1. Import the `dashboard.xml` file into your Splunk instance.
2. Load your dataset (or use any login failure logs).
3. Dashboard will auto-populate with the visualizations.

The dashboard was created using a sample dataset for failed login attempts.
